# Plain HTML D&D Character Generator

This folder contains a standalone static-site version of the generator.

## Files

- `index.html`
- `style.css`
- `script.js`

## Deploy to Netlify

1. Open the `plain-html` folder.
2. Confirm `index.html` is inside the root of that folder.
3. Drag the whole folder into Netlify Drop, or upload it as a static site.

## Publish result

When deployed, visiting the site root will load `index.html` and run the generator as a plain HTML page outside WordPress.
